SSN reveal latency — direct DB vs API-fronted

Does an API tier in front of the database add latency a user would notice? Measured two ways, not guessed.
Server-side, the API tier costs ~16–19 ms. Add a real browser over the network and a ~45 ms floor appears — yet the worst path's p95 is still ~122 ms. Either lens: a fraction of a second. Nothing approaches "seconds."
